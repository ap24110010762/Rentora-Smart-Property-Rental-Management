# Rentora - Smart Property Rental Management System 

**Rentora** is a premium full-stack property management platform designed to simplify the rental process for both landlords and tenants. Built with the **MERN Stack**, it features a sophisticated "Nude Brown" aesthetic and smart relational logic.

## Key Features
- **Role-Based Access:** Distinct dashboards for Property Owners and Renters.
- **Smart Logic:** Landlords manage only their listings; Tenants see a global marketplace.
- **Maintenance Tracking:** Tenants can select a property to report specific issues to owners.
- **Rent Tracking:** Real-time logging of payments with detailed transaction history.
- **Search & Filter:** Instant city-based search for easy property discovery.
- **Secure Auth:** JWT session management and Bcrypt password hashing.

## Tech Stack
- **Frontend:** React.js, Axios, CSS3 (Outfit Font)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas (Cloud)
- **Architecture:** Model-View-Controller (MVC)
